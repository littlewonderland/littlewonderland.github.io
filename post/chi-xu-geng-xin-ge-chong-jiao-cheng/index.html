1.如何搭建属于自己的博客

不会代码，如何零成本搭建个人博客？——安哥拉

按照这篇文章说的做就可以，需要注意的是，在Gridea中录入库的信息时需要CNAME，据我摸索，CNAME填自己的博客网址就行，比如我就只需要填littlewonderland.github.io

关于图床：图床设置-Github图床 中的非必填项可以直接空着，我按文章作者的方法填了，但是用不了，然后我干脆把非必填的两项空着，立马就能用了。

2.如何给自己的博客添加评论区

方法一：通过代码直接装

为博客添加Gitalk评论插件——Alanzj
我实在整不明白代码应该夹在哪，所以没有用这个方法。后来发现这篇文章没讲申请OAuth……
所以建议先完成方法二中的第三步，取得 Client ID 和 Client Secret 后，修改代码再使用。

方法二：我没有走完其中的每一步，但还是做好了

Gitalk评论插件使用教程——Dragon

首先进行第三步，获取OAuth application，然后保存好 Client ID 和 Client Secret
再将 Client ID 和 Client Secret 填入Griea的 远程-评论设置（勾选Gitalk）。
我个人建议新建一个仓库，填新的仓库名称。
Owner就是你用于存评论的仓库的账号用户名。
同步之后，打开博客还不能用评论区，要求你搞一个issue，这个过程好像叫初始化。我也没搞懂，但是连蒙带猜瞎填，保存确认好了就可以用了。
大家自己摸索吧（溜走.gif）
